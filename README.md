# ARAM Group

Static Bootstrap website for ARAM export business. Start with [index.html](index.html), then navigate to the product, service, and process pages.

## Pages
- `index.html` - Home page
- `products.html` - Products page
- `service.html` - Services page
- `process.html` - Process page

## Tech Stack
- HTML
- CSS
- Bootstrap
- JavaScript

## How to Run
1. Open `index.html` in a browser.
2. Or use VS Code Live Server for local preview.

## Project Notes
- The site uses a contact drawer form on multiple pages.
- Contact form submissions are set up to work with Google Sheets through Apps Script.
- See `GOOGLE_SHEET_SETUP.md` for setup details.

## Folder Structure
- `css/` - Stylesheets
- `js/` - Bootstrap and custom scripts
- `image/` - Site images
